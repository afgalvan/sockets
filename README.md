# sockets

A socket practice with python as client and node as server

## Test the project

**Clone the repo**

```bash
git clone https://github.com/afgalvan/sockets.git
# Go to directory
cd sockets
```

**Install the dependencies**

```bash
# node
yarn
# python
make dependencies
```

**Run it**

- server

  ```bash
  yarn start <port>
  ```

- client

  ```bash
  make run port=<port> # Should be the same that server
  ```
