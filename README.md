# sockets
A socket practice with python as client and node as server

## Test the project

**Clone the repo**
```bash
git clone https://github.com/afgalvan/sockets.git
```

**Install the dependencies**
```bash
# node
yarn
# python
make dependencies
```

**Run it**
```bash
# server
yarn dev <port>
#client
make run port=<port>
```
